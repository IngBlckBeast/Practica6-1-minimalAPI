# Configuración de GitHub Actions runner

## Descripción
Creación de runner e instalación como servicio (para actividad independiente)

## Recursos utilizados
- Nuestro propio servidor 
- Activación de Runners desde GitHub

## ¿Qué se consiguió?
- Conseguimos crear nuestro runner BestiaParda en GitHub para nuestro repositorio, al cual se le dió una configuración como servicio para dejarlo independiente de incializaciones desde terminal.
- Al configurarlo, se agregaron labels como: linux, self-hosted, devops-class y docker.
